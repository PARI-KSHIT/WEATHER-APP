# WEATHER-APP
A responsive weather application built with HTML, CSS, and vanilla JavaScript. It fetches real-time temperature, humidity, and wind data using an external Weather API. Users can search for current conditions in any city. This project showcases API integration and dynamic UI updates.  Tech HTML5 | CSS3 | JavaScript

This is a simple, modern Weather Application that allows users to instantly check the current weather conditions for any city around the world.

The application fetches real-time weather data from a reliable external Weather API and displays key information such as temperature, humidity, wind speed, and a descriptive weather summary (e.g., "Clear Sky," "Partly Cloudy").

‚ú® Features
Real-Time Data: Fetches up-to-the-minute weather information.

Search Functionality: Easily search for the weather of any city globally.

Clean and Responsive UI: Designed for an intuitive user experience on both desktop and mobile devices.

Dynamic Backgrounds: Displays different visual icons or backgrounds based on the current weather condition (e.g., a sun for clear, a cloud for cloudy).

üöÄ Getting Started
Follow these simple steps to get a local copy up and running:

Prerequisites
You only need a modern web browser to run this application.

Installation
Clone the repository:

Bash

git clone https://github.com/PARI-KSHIT/WEATHER-APP.git
Navigate to the project directory:

Bash

cd weather-app
Open the file: Simply open the index.html file in your preferred web browser.

Note: If you are using a public weather API, you might need to insert your own API key in the JavaScript file to make the data fetching work. Look for a variable named something like API_KEY in the script.js file.
